# Email Campaign System - Backend

Golang-based backend for the email campaign management system.

## Features

- Multi-provider email sending (Gmail, Yahoo, Outlook, iCloud, Workspace)
- Smart account rotation with health monitoring
- Template system with spam detection
- 50+ personalization variables
- Proxy support (HTTP/HTTPS/SOCKS5)
- Telegram notifications
- Real-time WebSocket updates
- Rate limiting and throttling

## Requirements

- Go 1.21+
- PostgreSQL 15+
- Redis 7+
